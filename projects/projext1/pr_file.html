The Grand Idea

There are many requirements initiating development: 1st to sketch an idea/design, to code it and to publish a web-site (as well as steps taken for the most crucial w/security level as considered).

In Codecamy's skillpath lessons (my choice) the core fuctionality is provided by JavaScript(JS): the lessons proceed

through the basics of writing JS-code, also for its purpose to bring about functionality to a published web-page.

JS-code can get called from the code 1st read into a browser. Interpreting of code files is based on 'stacks' (eg. NGINX) built on top of the web services. The initial landing page a browser opens, typically index.html and this is where the call get read. HTML code that calls JS code is simple (1-line of code).
If the development harnesses multiple Front End development technology to be published from sub pages of web-page site, then
this requires learning coding in the introduction to node.js and how it can be set up locally.

So beside Node, for instance, local test enviroment is made possible by Node. It involves setting up .nvm, a local environment designed for this development purpose.
